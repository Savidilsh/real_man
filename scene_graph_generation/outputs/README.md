# Outputs

Pipeline run folders are written here by convention.

Important output files:

```text
04_stitched/scene_graph.json
04_stitched/scene_graph_detailed.json
04_stitched/floor_instances.json
04_stitched/floor_instances.npz
```

After interactive editing, use the latest saved session:

```text
05_interactive_edits/session_*/scene_graph.json
05_interactive_edits/session_*/scene_graph_detailed.json
05_interactive_edits/session_*/floor_instances.json
05_interactive_edits/session_*/floor_instances.npz
```

These outputs are ignored by git.
